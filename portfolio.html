01 핵심 요약

맥락 없는 AI 분석은, 속도만 빠른 착각이다

AI에게 마케팅 데이터를 주면 빠르게 분석해 준다. 그러나 '왜'를 모른 채 내리는 판단은, 분석이 아니라 오판의 자동화다. 인게임 이벤트로 ROAS가 급등했을 때 "예산을 증액하세요"라고 할 것이고, 플랫폼 정책 문제로 지출이 급감했을 때 "매체를 이동하세요"라고 할 것이다.

본 기획서는 AI 에이전트가 매체 API, 캘린더 등에서 운영 맥락을 자동 수집하고, 파악되지 않는 맥락만 담당자에게 확인하여 신뢰할 수 있는 의사결정을 돕는 Context-Aware Marketing AI Agent를 제안한다.

02 문제 정의

왜 지금의 AI 마케팅 분석은 위험한가

LLM은 숫자만 본다 — 맥락 없이 분석하면 틀린다

마케팅 데이터의 변동에는 항상 이유가 있다. 인게임 이벤트, 매체 세팅 변경, 플랫폼 정책 이슈, SDK 장애 등 — 숫자 뒤에 숨은 원인을 모른 채 분석하면 필연적으로 오판한다.

실제 상황LLM의 오판실제 원인
인게임 페이백 이벤트로 ROAS 급등 "성과 개선! 예산 증액!" 이벤트 종료 시 원상복귀. 증액하면 예산 낭비
구글 정책 위반으로 광고 블락 "Google 급감! Meta로 이동!" 정책 이슈. 이의제기하면 복구됨
소재 교체로 일시적 CPI 상승 "효율 급락! 기존 소재 복원!" 의도적 교체. ML 재학습 기간의 일시적 변동
SDK 장애로 iOS 이벤트 수집 지연 "iOS 설치 급감! CPI 폭등!" SDK 지연으로 인한 일시적 집계 오류

공통점: 데이터에는 '무엇이' 변했는지만 있고, '왜' 변했는지는 없다.

데이터는 넘친다 — 그러나 담당자는 한두 명이다

AppsFlyer, 매체 대시보드, 내부 BI — 데이터 소스는 넘치지만 추출, 해석, 인사이트 도출은 1~2명의 담당자에게 집중된다. AI 분석이 늘어날수록 검증할 여력은 줄고, 잘못된 분석이 걸러지지 않고 의사결정에 반영될 위험이 커진다.

AI만 믿으면 판단을 그르치고, 사람에게만 의존하면 한계가 있다.
AI가 맥락을 이해하되, 최종 판단은 사람이 하는 구조가 필요하다.

03 솔루션

맥락을 주고, 판단을 검증한다 — 두 축으로 해결

3.1 맥락 수집 시스템 (Context Layer)

매체 API, 캘린더, 장애 로그 등에서 자동 수집 가능한 맥락은 시스템이 먼저 수집하고, 자동으로 파악되지 않는 맥락(변경 이유, 외부 요인 등)만 담당자에게 확인한다.

L1
도메인 지식 (영구)
UA 스페셜리스트 에이전트의 판단 기준. CPI/ROAS/NetROAS 해석, 인과관계 판단, 위험 시그널 인식, 런칭/서스테이닝 모드 전환.
L2
Context Layer (매일 갱신)
4개 서브시스템에서 자동 수집된 맥락 데이터
L3
성과 데이터 (API 연동)
AppsFlyer MCP, 매체 대시보드, 내부 BI에서 실시간 수집

Context Layer 4개 서브시스템:

📝

01 UA운영 변경 이력

매체 API로 예산/소재/타겟팅 변경 자동 수집. 변경 이유만 보완 질문.

📅

02 이벤트 캘린더

인게임 이벤트/프로모션 일정 자동 동기화. 미등록 이벤트만 보완 질문.

🚧

03 정책/장애 로그

플랫폼 정책 변경, SDK 장애 자동 모니터링. 미감지 이슈만 보완 질문.

🔌

04 이상치 자동 탐지

통계 기반 이상 감지 후 01~03과 원인 매칭. 미매칭 시 즉시 담당자에게 질문.

같은 데이터, 다른 판단:

Context 없이

"ROAS 42% 급등! 예산 증액하세요!"

Context Layer 적용

"ROAS 42%는 페이백 이벤트 영향입니다. 실질 ROAS는 약 20%입니다. 이벤트 종료(3/7) 후 재평가하세요."
신뢰도 LOW 이벤트 기간 겹침

3.2 AI 판단 검증 장치 (가드레일)

AI의 판단에 자신감 수준을 강제 표기하여, 사람이 검증 없이 수용하는 것을 구조적으로 방지한다.

08 우리의 접근

AI에게 무엇을 맡기고, 무엇을 쥐고 있을 것인가

INSIGHT 1 — AI를 잘 쓴다는 것

많이 쓰는 것도, 빠르게 쓰는 것도 아니다. 무엇을 맡기고, 무엇을 내 손에 쥐고 있어야 하는지 아는 것이다.
→ Human-in-the-Loop. 이 경계를 시스템으로 구현한 것이 가드레일이다.

INSIGHT 2 — 블랙박스 시대에 남는 것

에이전트가 데이터를 처리한다. 하지만 어떤 데이터가 중요한지는 사람이 안다. 남는 건 목표를 정의하는 사람, 맥락을 아는 사람이다.
→ Context Layer. 현장 전문가의 맥락 지식을 시스템에 녹인다.

INSIGHT 3 — AI는 사고를 증폭하는 파트너

AI는 정답을 주는 기계가 아니라, 사고를 증폭하는 파트너다. 맥락(Context)이 영혼을 만든다.
→ 같은 ROAS 42%도 "페이백 이벤트 기간"이라는 맥락이 붙는 순간 완전히 다른 의미가 된다.

이 시스템은 사람을 대체하지 못한다.
다만, 사람의 판단을 더 빠르고, 더 정확하게 만드는 것 — 그것이 이 시스템의 방향이다.
